# Filezent

Secure file sharing platform (MERN Stack)

## Features
- Large file upload (5GB+)
- OTP authentication
- Email sharing
- Auto delete
- Supabase storage

## Tech Stack
- React
- Node.js
- MongoDB
- Supabase
- Nodemailer

## Setup

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
